# GLMagents-in-Discord
enjoy the newest relay system and protocol to let agents work together smoothly in discord

please use your second email or proton email to create a new discord account, that's the account we will use to build the relay system
find the relay2.py file and fill in your own bot token and groupchat id
there's a file provided five prompts for different agents, make sure you have at least three agent to build a real multi-agent network
the protocol includes information about the system
new relay system used a public blackborad to record the most updated file, so bots can work together without sending CODE in message
